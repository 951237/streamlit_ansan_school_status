2025-11-09 개선사항 목록

1. 자동 데이터 수집 파이프라인 구축
   - 대상 URL: https://www.goeas.kr/goeas/cm/cntnts/cntntsView.do?mi=11528&cntntsId=1514 페이지의 표 데이터를 스크레이핑한다.
   - GitHub Actions 또는 로컬/서버 cron에서 `python crawl_data.py`를 하루 1회 실행해 최신 CSV를 생성하고, 정합성 검사 후 커밋·푸시까지 자동화한다.

2. Streamlit Cloud 휴면 방지
   - UptimeRobot 같은 무료 모니터링 서비스로 5분 간격 헬스체크를 걸어 앱이 휴면 상태로 떨어지지 않도록 한다.

3. 상시 접근 강화를 위한 배포 가이드
   - 기존 Streamlit Cloud를 기본으로 유지하되, 필요 시 Render/Railway/Fly.io 등의 무료 PaaS로 컨테이너 배포할 수 있도록 `streamlit run app.py --server.headless true --server.port $PORT` 기반의 Procfile 및 Dockerfile 예시를 준비한다.

4. 대시보드 편의 기능
   - 최종 데이터 수집 시각 배지, 학교급 필터/검색, 주요 지표 스냅샷을 상단에 노출해 재방문자가 변경사항을 빠르게 인지할 수 있도록 UI를 개선한다.

5. 데이터 확장 및 시각화 고도화
   - 학교 좌표를 확보해 지도 시각화를 추가하고, 학급당 학생 수 등 파생 지표를 계산해 의사결정을 돕는 섹션을 멀티 페이지 구조(개요/학급수/교원현황 등)에 배치한다.
